```
Chicago Crime Analysis Web App
```

```
Required Javascript packages:

- express
- node
- nodemon (optional)
```

```
How to run the application:

1. Install node.js on your local machine:
  https://nodejs.org/en/download/

2. Clone the repository into your local machine:
  git clone https://github.com/Vismark1994/ChicagoCrimeAnalysisWebsite.git

3. cd into the cloned repository:
  cd ChicagoCrimeAnalysisWebsite

4. run the following command to start the node server on port 5000:
  node server.js

5. Open a browser and navigate to: http://localhost:5000

6. You may access any other route listed in the server.js file
```
